# docker-deploy-bsr
> Docker deploy script for browser side rendering project.

## install:
```shell
cd /data/my-project
git clone https://github.com/afeiship/docker-deploy-bsr.git deploy
```

## development:
```shell
cd /data/my-project
./deploy/dev/start.sh
```


## production:
```shell
cd /data/my-project
./deploy/prd/start.sh
```
